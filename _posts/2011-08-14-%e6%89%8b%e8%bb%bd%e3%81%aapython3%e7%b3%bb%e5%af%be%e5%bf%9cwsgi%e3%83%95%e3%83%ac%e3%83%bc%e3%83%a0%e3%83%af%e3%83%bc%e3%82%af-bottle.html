>bottleというフレームワークが素晴らしくシンプルだったので、書いてみる。

まずはインストール。
[bash]pip install bottle
[/bash]
適当なフォルダに以下のファイル作成。
test.py
[python]from bottle import route, run

@route('/hello/:name')
def index(name='World'):
return '<b>Hello %s!</b>' % name

run(host='localhost', port=8100)
[/python]
あとは実行するだけ。
[bash]python test.py
[/bash]
http://localhost:8100/hello/matsu で以下の画面が出る。

超簡単！
