今回railsで早くサービスを創り上げるために僕がやったいくつかの手順。

rails new myGems -d mysql

vi Gemfile

以下を追記

assetsに

gem "twitter-bootstrap-rails"

gem 'less-rails'

それ以外に

gem 'rails-backbone'

gem 'bcrypt-ruby'

gem 'unicorn'

gem 'capistrano'

gem 'omniauth'

gem 'omniauth-twitter'

group :development, :test do

#spec

gem 'rspec'

gem 'rspec-rails'

gem 'spork'

gem 'guard'

gem 'guard-spork'

gem 'guard-rspec'

gem 'guard-test'

gem 'growl'

gem 'growl_notify'

end

bundle update

capify .

rails g bootstrap:install

rails g backbone:install

rails g rspec:install

rails g model User name:string screen_name:string provider:string uid:string token:string token_secret:string

rails g model gemdata title:string viewed:integer

rails g model comment description: string fav:integer user:belongs_to gemdata:belongs_to

rails g model tag title:string

rails g model using user:belongs_to gemdata:belongs_to

migrationファイル書き換え

controller作成

rails g controller session callback logout
